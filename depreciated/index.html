Research Experience


Rack-Based Dynamic Workload balancing in Hadoop System (August 2016-December 2017)
  • Designed an algorithm which accurately measures the workload of the racks in a heterogeneous Hadoop cluster and reduces the runtime of map tasks by 28%.
  • Implemented load detection framework which feeds on the YARN log files, topology files, and other custom logs to calculate metrics to describe the workload of racks.
  • Developed a pluggable scheduler which works in with the Native Hadoop Scheduler to modify the scheduling of the map tasks to each rack.
  • Modified Hadoop 2.0 source code to implement the scheduling part of the algorithm.
  • Configured a heterogeneous cluster on Amazon EC2 environment using the concept of rack awareness.
  • Demonstrated the robustness of the algorithm by comparing the performance Hadoop Source modified with the algorithm with native Hadoop source.
